# 🚀 Antigravity AI - Runner's Guide

Follow these steps to get your premium AI Chatbot SaaS up and running in minutes.

## ⚡ Quick Start (Run Both Together)
From the root folder, you can start both the client and server with one command:
```powershell
npm run dev
```

---

## 🛠️ Step 1: Start the Backend
The backend handles authentication, database storage, and the AI bridge.

1. Open a terminal and navigate to the `backend` folder:
   ```powershell
   cd backend
   ```
2. Run the server in development mode:
   ```powershell
   npm run dev
   ```
   *Expected Output: `MongoDB Connected` and `Server running on port 5050`.*

---

## 🎨 Step 2: Start the Frontend
The frontend provides the premium SaaS user interface.

1. Open a **new terminal** and navigate to the `frontend` folder:
   ```powershell
   cd frontend
   ```
2. Start the Vite development server:
   ```powershell
   npm run dev
   ```
   *Expected Output: `VITE v... ready in ...ms`.*

---

## 🌐 Step 3: Access the App
Once both are running:
- Open your browser and go to: **[http://localhost:3000](http://localhost:3000)**
- Click **Join Us** or **Get Started** to create a test account.
